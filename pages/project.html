Nyxtext Zenith : Zenith of Editing

A modern, keyboard-centric text editor built with QT and QScintilla, Supports 35+ programming languages out of the box and features Lua-based customizations for themes and keybindings. It offers a sleek interface with the Tokyonight theme by default, code folding, and a dockable file tree. It's designed to be highly efficient for daily coding tasks while remaining easily extensible.

NyxText: A Versatile Text Editor for Writers and Coders

Nyxtext is a text editor built with customtkinter and tkinter to deliver a seamless writing and coding experience. This versatile tool provides basic text editing by offering an intuitive file tree for effortless project navigation and an integrated terminal for swift command execution. With over a dozen carefully crafted themes spanning both dark and light modes, Nyxtext caters to every visual preference while maintaining a sleek, modern interface. The editor boasts a rich set of features including syntax highlighting, customizable shortcuts. Currently depreciated (Check Nyxtext Zenith)

Arch Workflow - i3wm, tmux, neovim & vscode

A minimalist Arch setup for a keyboard-centric distribution. The workflow consists of i3wm, polybar, tmux, and neovim. The configuration is designed to be lightweight and efficient, with the goal of maximizing productivity. The setup is optimized for speed and ease of use, with a focus on keyboard shortcuts and automation. The dotfiles are highly customizable and can be tailored to suit individual preferences. The workflow is ideal for developers, sysadmins, and power users who value efficiency and simplicity. The dotfiles are open-source and available on GitHub for anyone to use or contribute to. The project is actively maintained and updated regularly with new features and improvements.
